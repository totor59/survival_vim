SURVIVAL GUIDE

LA BASE DE VIM
Commandes de bases
i Mode insertion
$ Aller en fin de ligne
:q Quitter
:q! Quitter sans enregistrer
:wq Quitter et enregistrer
w Enregistrer
^v Mode visuel
Echap Sortir du mode insertion
Déplacements
h,j,k,l Déplacer le curseur
w,b Avancer ou reculer d'un mot
gg Aller en début de fichier
G Aller en fin de fichier
Edition
dd Couper la ligne
x Effacer caractère sous curseur
diw Effacer le mot
yy Copier la ligne
p Coller après p
y(en mode visuel) Copier la sélection
Recherche/remplacements
/ Rechercher
:%s/A/B/g Remplace tous les "A" par des "B"
Mise en page
gg=G(en mode visuel) Indente tout le fichier
Fenêtres
:vsp nomdefichier Ouvre un fichier en split vertical
^w Naviguer entre les splits
Onglets
:tabnew nomdefichier Ouvre un fichier dans un nouvel onglet
gt Naviguer entre les onglets
